# Python Learning Journey 🐍

Repositorio personal donde voy subiendo ejercicios, apuntes y mini proyectos mientras aprendo Python.

## Objetivo

- Aprender Python desde cero de forma práctica.
- Reforzar conceptos como variables, condicionales, bucles y funciones.
- Crear una base sólida para automatización y análisis de datos.

## Estructura del repositorio

- `functions/` → ejercicios sobre funciones.
- `loops/` → ejercicios sobre bucles.
- `mini_projects/` → pequeños proyectos prácticos.
- `notes/` → apuntes y resumen de conceptos.

## Progreso

- Variables ✅
- Condicionales ✅
- Bucles ✅
- Funciones ✅
- Estructuras de datos ⏳
- Mini proyectos ⏳

## Cómo usar este repositorio

Cada archivo contiene un ejercicio o ejemplo práctico con código comentado para facilitar el aprendizaje.

Estoy aprendiendo Python como parte de mi camino en análisis de datos, automatización y eficiencia operativa.
