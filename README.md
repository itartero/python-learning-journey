# Functions Exercises

Esta carpeta contiene ejercicios y ejemplos prácticos relacionados con funciones en Python.

## Objetivo

Practicar el uso de funciones para escribir código más reutilizable, limpio y fácil de mantener.

## Contenido

- Definición de funciones.
- Parámetros y argumentos.
- Valores de retorno.
- Uso de `map()`.
- Uso de `lambda`.
- List comprehensions como alternativa a `map()`.

## Ejercicios incluidos

- Conversión de cadenas a enteros con `map()`.
- Suma de listas con `map()` y `lambda`.
- Cuadrados de números con list comprehensions.

## Aprendizajes

En esta sección estoy reforzando:
- cómo transformar datos de forma eficiente,
- cómo aplicar funciones anónimas,
- y cuándo usar `map()` o una list comprehension según el caso.
